# "ООО Обувь"

## Начало работы

### Подготовка окружения

Прежде всего потребуется установить необходимые библиотеки Python:

```bash
pip install -r requirements.md
```

### Запуск приложения

```bash
python main.py
```

## Учетные данные для входа

Для доступа к системе используйте следующие учетные записи:

| Роль | Логин | Пароль |
|------|-------|--------|
| admin | admin | admin123 |
| Менеджер | manager | manager123 |
| Покупатель | client1 | client123 |

## Права доступа по ролям пользователей

### Роль: Гость
Ограниченные возможности:
- Ознакомление со списком доступных товаров

### Роль: Авторизованный покупатель
Доступ покупателя:
- Просмотр каталога товаров

### Роль: Менеджер
Расширенные полномочия:
- Поиск и фильтрация товаров по различным критериям
- Поиск по названию и характеристикам
- Группировка товаров по поставщикам
- Управление и просмотр заказов

### Роль: Администратор
Полные права управления:
- Работа со всем функционалом менеджера
- Создание, изменение и удаление позиций в каталоге
- Формирование, редактирование и удаление заказов
- Загрузка фотографий и изображений售товаров
- Управление поставщиками
